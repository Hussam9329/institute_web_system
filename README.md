# Institute Web System

نظام ويب لإدارة المعهد، يهدف إلى تنظيم العمليات الإدارية والمالية داخل المؤسسة التعليمية من خلال واجهة سهلة الاستخدام، مع دعم إدارة الطلاب والمدرسين والتقارير وملفات PDF.

---

## 📌 Overview

يقدم هذا المشروع نظامًا إداريًا متكاملًا نسبيًا للمعاهد التعليمية، حيث يساعد على:

- إدارة بيانات الطلاب
- إدارة بيانات المدرسين
- متابعة الجوانب المالية والمحاسبية
- عرض الإحصائيات العامة
- إنشاء تقارير وملفات PDF
- تحسين تنظيم العمل الإداري وتقليل الجهد اليدوي

---

## ✨ Features

### الطلاب
- عرض قائمة الطلاب
- إضافة طالب جديد
- تعديل بيانات الطالب
- عرض الملف الشخصي للطالب

### المدرسون
- عرض قائمة المدرسين
- إضافة مدرس جديد
- تعديل بيانات المدرس
- عرض تفاصيل المدرس

### المحاسبة
- إدارة السجلات المالية
- دعم الإيصالات والتقارير
- تنظيم البيانات المالية الخاصة بالمعهد

### الإحصائيات
- عرض ملخصات وإحصائيات عامة للنظام

### التقارير وملفات PDF
- إنشاء ملفات PDF للطلاب
- إنشاء ملفات PDF للمدرسين
- إنشاء إيصالات وتقارير قابلة للطباعة

### الواجهة
- تصميم يدعم الاتجاه من اليمين إلى اليسار (RTL)
- ملفات CSS مخصصة للطباعة
- JavaScript منفصل لكل جزء من أجزاء النظام
